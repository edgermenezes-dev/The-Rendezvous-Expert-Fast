STEP 1: DOWNLOAD PACKAGE
[ ] Download rendezvous-godaddy-deploy.tar.gz to your computer
[ ] Verify file size is 163 KB

STEP 2: LOGIN TO GODADDY
[ ] Go to godaddy.com and sign in
[ ] Navigate to: My Products → Web Hosting → Manage
[ ] Click "cPanel Admin" button

STEP 3: OPEN FILE MANAGER
[ ] In cPanel, find "FILES" section
[ ] Click "File Manager"
[ ] Navigate to public_html folder

STEP 4: CLEAN EXISTING FILES
[ ] Backup old files (if needed)
[ ] Select all files in public_html
[ ] Click "Delete" and confirm
[ ] public_html should now be EMPTY

STEP 5: UPLOAD PACKAGE
[ ] Click "Upload" button in File Manager
[ ] Select rendezvous-godaddy-deploy.tar.gz
[ ] Wait for upload to complete
[ ] Close upload tab

STEP 6: EXTRACT FILES
[ ] Refresh File Manager (F5)
[ ] Right-click on .tar.gz file
[ ] Select "Extract"
[ ] Leave path as default
[ ] Click "Extract File(s)"
[ ] Click "Close"

STEP 7: DELETE ARCHIVE
[ ] Select rendezvous-godaddy-deploy.tar.gz
[ ] Click "Delete"
[ ] Confirm deletion

STEP 8: VERIFY FILES
[ ] Enable "Show Hidden Files" in Settings
[ ] Verify these files exist in public_html:
    [ ] index.html
    [ ] .htaccess
    [ ] manifest.json
    [ ] robots.txt
    [ ] service-worker.js
    [ ] static/ folder

STEP 9: SET PERMISSIONS
[ ] Select all files (not folders)
[ ] Click "Permissions"
[ ] Set to 644
[ ] Select static folder
[ ] Click "Permissions"
[ ] Set to 755

STEP 10: TEST WEBSITE
[ ] Visit https://yourdomain.com
[ ] Verify website loads
